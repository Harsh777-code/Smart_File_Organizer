# 📂 Smart File Organizer

A Python desktop application that automatically organizes files into categorized folders based on their file extensions.

---

## ✨ Features

- 📁 Browse and select any folder
- 📂 Automatically creates category folders
- 🖼️ Organizes Images
- 📄 Organizes Documents
- 🎬 Organizes Videos
- 🎵 Organizes Music
- 📦 Organizes Archives
- 🖥️ Modern GUI built with CustomTkinter

---

## 🛠️ Technologies Used

- Python 3
- CustomTkinter
- Tkinter
- os
- shutil

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Harsh777-code/Smart_File_Organizer.git
```

### Go to the project folder

```bash
cd Smart_File_Organizer
```

### Create a virtual environment (optional)

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```
